# descenders-teleport_menu
teleportation menu for descenders
<br><br>
1. Create an empty game object and add component TeleportMenu or drag the TeleportMenu.cs onto the game object and goto Inspector<br>
2. Define Toggle Key, preserve velocity or not, and set Size to amount of teleportation zones you want<br>
3. Drag a game object or prefab from your hierarchy to the element fields for each teleportation zone<br>
4. Export game - to resize the menu window, hit esc then you can use the mouse in lower right corner to change the window size.
